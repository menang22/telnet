# Telnet
Script kumpulan perintah-perintah telnet versi menang22

## Cara Penginstallan

$ pkg update && pkg upgrade

$ pkg install git nano

$ pkg install bash curl

$ pkg install python python2

$ pkg install busybox

$ git clone https://github.com/menang22/telnet

$ cd telnet

$ sh run.sh
